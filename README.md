Terraform EC2 Strapi Project

This project provisions an AWS EC2 instance using Terraform with a module-based structure.  
It also generates a PEM key, installs Node.js, and runs Strapi
Features

- EC2 Provisioning using Terraform  
- PEM Key Pair created & managed by Terraform  
- Module-based folder structure  
- Strapi installation using EC2 `user_data`   
- `.gitignore` configured 
 Used
- Terraform  
- AWS EC2  
- AWS Key Pair  
- Ubuntu 22.04  
- Git + GitHub
